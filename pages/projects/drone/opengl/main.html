Drone opengl sim

Overview

The C++ simulator is more bare-bones than the Python/Unity simulator, but it's more realistic in the physics that it models. The aim of the project is to:
  • Setup the C++ simulator and the development environment (from the python code)
  • Familiarisation with tools to evaluate the controller's performance
