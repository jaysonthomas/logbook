Classes

Bot

  • dir
    • Angle of the bot stored as a 2D vector.

Map

  • inPolygonFormat
    • Contains x, y coordinates of the map for use in functions like inpolygon (which checks if points are inside a polygon). The first point of the map is appended again to the end, closing the loop of the polygon by connecting the last vertex to the first.

    inPolygonFormatX, inPolygonFormatY contain all the lengths of the X-axis and Y-axis vectors respectively. These are also used in drawing the map with matplotlib's line function which require all the x and y vector lengths to be separate.

  • line
    • Stores each border line as [x1, y1, x2, y2].

  • point
    • Stores the closed map with the first point repeated at the end.

Matrix

  • createTranslation
    • Converts a 2D vector into a translation matrix with homogenous coordinates.

  • createZRotation
    • Converts an angle into a rotation matrix with homogenous coordinates.

Scanner

  • locationOffset
    • This is the physical location of the ultrasound sensor. The sensor is mounted on a motor. The centre of rotation would ideally be located at the centre of the bot from a top-down view. If there is an offset from the centre of the bot, then this is captured in this variable.

  • config
    • Stores how the bot performs a single scan:
    • The number of points/samples collected
    • The angle between samples

Particular variables

  • map
    • It is of type Point. The first column, map[i][0], contains the x coordinates, and the second column contains the y coordinates.
