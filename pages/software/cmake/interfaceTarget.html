Cmake Commands: Target

Library


add_library(matplotlib_cpp INTERFACE)
This creates a CMake INTERFACE target library called matplotlib_cpp. An interface library doesn't produce a binary or object files but defines properties like include directories, linked libraries, etc. that are applied to other targets that link against it.

Include Directories


target_include_directories(
  matplotlib_cpp
  INTERFACE
    $<BUILD_INTERFACE:${PROJECT_SOURCE_DIR}/examples>
    $<INSTALL_INTERFACE:include>
)
This sets up include directories for the target matplotlib_cpp, based on whether the project is being built or installed.

After installation, the include directory will be include, where headers like matplotlibcpp.h will be installed. When building, it includes the examples directory.

Compile features


target_compile_features(matplotlib_cpp INTERFACE cxx_std_11)
This specifies that any project linking against matplotlib_cpp must support C++11 (cxx_std_11). This ensures the correct C++ standard is used.
