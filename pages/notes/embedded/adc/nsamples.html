Setting the number of samples

Overview

To set the number of samples captured per ADC channel, DMA is used by setting DMA_CNDTRx depending on chip type.

The ADC is dumb; if it is set to continuous mode, then it will sample forever. To limit the number of samples, setup the DMA controller to only transfer the specific number of samples to a predefined buffer. When the DMA counter CNDTR reaches 0, the DMA stops transferring and triggers a "Transfer Complete" interrupt. The ADC can then be stopped in that interrupt.
