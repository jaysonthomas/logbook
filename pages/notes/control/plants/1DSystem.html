1D System

Overview

1D does not refer to a 1D mechanical system, but a 1D state space, i.e. the state is scalar, $x \in \mathbb{R}$.

Non-linear systems are hard and there are too many unknowns. However, everything about a 1D non-linear system can be analysed.

Equilibrium points are at $\dot{x} = f(x) = 0$.

Example: $\dot{x} = x^3$

Stability analysis

Considering the equilibrium point at the origin, if the system is given a very small nudge to the right or the left, from the sign of $\dot{x}$ (negative and positive respectively), it can be seen that the system would return to the origin. For small perturbations to the system at the origin, it returns back to the origin. It is a stable point.

Considering the rightmost equilibrium point; for a small nudge to the left or right, the system is going to move farther to the left and right respectively. It shoots off from the equilibrium point.

Every state within the 2 outer unstable points is inside the basin of attraction. Every state outside of this region will blow up to infinity.
