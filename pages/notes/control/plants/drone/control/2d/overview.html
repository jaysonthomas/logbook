2D Drone control: Overview

A quadrotor is symmetric. Therefore, the forward-backward motion is very similar to the left-right motion.

Underactuation
  • A 2D planar quadrotor is an underactuated system because while it has 3 DOFs, with its $y, z, \phi$ coordinates, only 2 quantities, thrust $u_1$ and rotational moment $u_2$, can be controlled.

  • At any given time, only 2 of those 3 DOFs can be controlled by sacrificing control of the third.

Linearisation
  • To control $y, z$, if the dynamics of the system is linearised around the hover state, equations that govern $y, z$ become decoupled, allowing them to be controlled independently. $u_1$ can be used to control altitude and $u_2$ can be used to control lateral motion in the y direction.
Cascaded control

  • In order to control motion in the y direction, $u_2$ would first need to control the intermediate variable of roll $\phi$.

  • This leads to the use of a cascaded control structure where the inner faster control loop which controls the roll angle is first implemented and tuned. Then the slower outer control loop is implemented which uses the y trajectory to set the target roll angle for the inner controller.
