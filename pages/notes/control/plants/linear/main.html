State transition function

[Ref]

$$\begin{align*} z_k &= Cx_k \\ x_{k+1} &= Ax_k + Bu_k \end{align*}$$
$$\begin{array}{ccl} A &:& \text{System matrix - it maps how the previous state relates to the next} \\ B &:& \text{Input matrix - it maps how the control input relates to the next state} \\ C &:& \text{Output matrix - it maps a state to the measurement space} \end{array}$$
Considering a non-simulation environment, at time instant $k$, a prediction $z_k$ can be made of the actual measurement $y_k$ that would be collected of the state at the time instant. In a simulation environment, this would serve as the measurement.

Once, the control input to be applied $u_k$ is decided based on the current state measurement, a prediction can be made of what the state would look like in the next time instant, $x_{k+1}$.

$y_k$, $z_k$ point to the state variables that the controller aims to control to follow a desired trajectory. It is not necessarily the observed output, that is, it is not necessarily directly measurable.
