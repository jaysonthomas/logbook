Overview

Equilibrium points are those with $\dot{x} = 0$.

The way to check for stable/unstable equilibria, is to take the first derivative of the dynamics w.r.t. x, $\frac{\partial{f}}{\partial{x}}$, which corresponds to the $A$ matrix in a linear system. If all of the eigenvalues of $A$ have a strictly negative real part, the system is stable. If any of the eigenvalues are 0, the stability is inconclusive and fancier analysis tools would need to be used.

The eigenvalues correspond to eigenvectors. There are particular eigenvector directions along which a system can be stable in some cases or unstable. If a system has a mixed spectrum of stable and unstable eigenvectors, even the slightest noise perturbation along an unstable eigenvector can cause the system to blow up to infinity. It is possible to do a more complex analysis and compute stable/unstable manifolds around equilibria of where a system can evolve.
