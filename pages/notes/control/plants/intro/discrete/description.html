Description

In a discrete-time system, the state at the next time step $k+1$ is a function of the current state $x_k$: $$ x_{k+1} = f_d(x_k) $$ The dynamics can also be written as an iterated map - it is a sequence of states rather than a smooth curve. $$ x_N = f_d(f_d(f_d(\dots f_d(x_0) \dots))) $$ State at time $N$, $x_N$ is found by plugging the starting point $x_0$ into the dynamics function $f_d$. The result is then plugged back into the same function over and over.
