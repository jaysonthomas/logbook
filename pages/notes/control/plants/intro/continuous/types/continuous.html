Continuous time smooth system

The most general form for a smooth system is a first order ODE: $$ \dot{x} = f(x, u) $$
$$\begin{array}{ccl} x &:& \text{State} \in \, \mathbb{R}^n \text{(vector of size } n \text{)}\\ f &:& \text{Dynamics (here, a function)} \\ u &:& \text{Control input} \in \, \mathbb{R}^m \\ \dot{x} &:& \text{Time derivative of the state vector} \end{array}$$
Any mechanical system can be written as a first order ODE by making the state vector twice as big: $$ x = \begin{bmatrix} q \\ v \end{bmatrix} $$
$$\begin{array}{ccl} q &:& \text{Configuration/joint angles, in physics. In robotics, it is referred to as pose. It is not always a vector.} \\ v &:& \text{Velocity. It is always a vector.} \end{array}$$
$q$ does not always live in the vector space $\mathbb{R}^n$ since orientations don't add like vectors. Roll, then pitch is not the same as pitch then roll.

An example of a smooth continuous time system is a simple pendulum.
