Equilibria

Overview

Equilibrium point of a dynamical system is a point at which a system is at rest, i.e. there is no movement; if it is placed there, it will remain there. $$ \dot{x} = f(x, u) = 0 $$ From an algebraic perspective, an equilibrium point is a point where the derivative is a root or a zero of the dynamics.

Moving an equilibrium point

To move an equilibrium point to a desired state $x^*$ using control, it would be necessary to find a value of $u$ for which $\dot{x} = 0$ at $x^*$. It boils down to a root-finding problem in (solving for) $u$ : $$ f(x^*, u) = 0 $$
  • X axis: $u$
  • Y axis: $\dot{x} = f(x^*, u)$
Simple pendulum example.

Controllability

It is not possible to do this when there are more joints or DOFs than actuators, i.e. in an underactuated system. In such a case, it would be necessary to reason about the dynamics in a sophisticated way since it is not possible to cancel out torques arbitrarily.
