Lower dimensional system

Overview

A system is stable if (this is the desirable case when designing a control system): $$ \frac{\partial{f}}{\partial{x}} < 0 $$ A system is unstable if: $$ \frac{\partial{f}}{\partial{x}} > 0 $$ The slope condition is a local condition based on a taylor expansion argument, which is a linearisation argument.
