Overview

Definitions

There are multiple definitions of stability. Asymptotic or exponential stability is an intuitive definition that checks whether a system will stay near (in the neighbourhood of) an equilibrium point under small perturbations.

A basin of attraction is the set of the state space from where if a system started, it would end up at an equilibrium point.

A saddle point $\frac{\partial{f}}{\partial{x}} = 0$, is only seen from a 2D system and upwards; example of a parabola.

Stability analysis via linearisation

To analyze the stability of a nonlinear system at an equilibrium point $x^*$, the following steps are taken:
  • Solve $f(x) = 0$ to find the equilibrium points $x^*$.

  • Calculate the general Jacobian i.e. the matrix of partial derivatives, i.e. the first derivative of the dynamics w.r.t. x, $\frac{\partial{f}}{\partial{x}}$.

  • Evaluate the jacboian at the equilibrium point of interest by plugging the equilibrium point $x^*$ into the Jacobian to get a matrix of constant numbers, $J(x^*)$.

  • Calculate and check the eigenvalues ($\lambda$) of $J(x^*)$.

Eigenvalues check

Eigenvalues correspond to eigenvectors. There are particular eigenvector directions along which a system can be stable in some cases or unstable. If a system has a mixed spectrum of stable and unstable eigenvectors, even the slightest noise perturbation along an unstable eigenvector can cause the system to blow up to infinity. It is possible to do a more complex analysis and compute stable/unstable manifolds around equilibria of where a system can evolve.

Core concept

[gemini]

The eigenvalue concept is based on the fact that, near an equilibrium point:
  • $\dot{x} = 0$
  • the dynamics can be locally approximated by $Ax$
If the system is not at an equilibrium point, then $f(x) \neq 0$. This doesn't hold up.

Doing a taylor expansion, $f(x) + A \Delta{x} \cdots$, at non-equilibrium setting, the first leading affine term is non-zero, so the expansion doesn't hold much information. There are things (not covered here) that can be said about stability along trajectories where the leading term is non-zero.
