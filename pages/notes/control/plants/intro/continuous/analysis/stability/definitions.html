Definitions

There are multiple definitions of stability. Asymptotic or exponential stability is an intuitive definition that checks whether a system will stay near (in the neighbourhood of) an equilibrium point under small perturbations.

A basin of attraction is the set of the state space from where if a system started, it would end up at an equilibrium point.

A saddle point $\frac{\partial{f}}{\partial{x}} = 0$, is only seen from a 2D system and upwards; example of a parabola.
