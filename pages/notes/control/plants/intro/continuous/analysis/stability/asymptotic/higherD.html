Higher dimensional systems

Background

$\frac{\partial{f}}{\partial{x}}$ is no longer a scalar, but a Jacobian matrix. If $\dot{x} = f(x)$ and $x \in \mathbb{R}^n$, then the Jacobian matrix $J = \frac{\partial{f}}{\partial{x}}$ is a square matrix ($n \times n$). See matrix shape.

Stability criteria

For a symmetric matrix, the eigenvalues are all pure real. In most dynamical systems, the Jacobian is a non-symmetric matrix, for which the eigenvalues can be complex.

For stability, the real parts of the eigenvalues of the Jacobian matrix need to be strictly negative.

Physical meaning

The imaginary parts correspond to oscillatory behaviour. As long as the real parts which correspond to damping are strictly negative, those oscillations will get damped out.

The more negative the values are, the quicker the system will return to an equilibrium point. The actual quantitative value depends on the choice of coordinate system (mm vs cm, s vs ms for example). But, if the eigenvalue spectrum of the system was available for a given coordinate system, this statement holds true considering the relative values.

Intuition

In general, a set of coordinates can be chosen for a dynamical system such that the dynamics decouple into a bunch of individual scalar systems. Each scalar system can then be treated as a 1D case.

If a system is linearised around an equilibrium, the eigenvectors define a coordinate transformation that decouples the dynamics into independent scalar systems, each governed by a single eigenvalue.

Steps to analyse stability

Take an eigen decomposition, i.e. decouple into $n$ 1D systems. $$\begin{align*} \text{Re}\left[\text{eigvals}\left(\frac{\partial{f}}{\partial{x}}\right)\right] < 0 &\implies \text{stable} \\ \text{otherwise} &\implies \text{unstable} \end{align*}$$ If there are any non-negative eigenvalues, the system is considered to be unstable.

Eigenvalues check

Eigenvalues correspond to eigenvectors. There are particular eigenvector directions along which a system can be stable or unstable.

If a system has a mixed spectrum of stable and unstable eigenvectors, even the slightest noise perturbation along an unstable eigenvector can cause the system to blow up to infinity. It is possible to do a more complex analysis and compute stable/unstable manifolds of where a system can evolve around an equilibrium.

Core concept

The eigenvalue concept is based on the fact that, near an equilibrium point:
  • $\dot{x} = 0$
  • the dynamics can be locally approximated by $Ax$
If the system is not at an equilibrium point, then $f(x) \neq 0$.

at a non-equilibrium point, a taylor expansion, $f(x) + A \Delta{x} \cdots$ consists of a leading affine term that is non-zero. $f(x)$ dominates the system. The Jacobian $A$ still contains information about the local stretching or shearing of the flow. In fluid dynamics, this is used even when not at equilibrium (look up Lyapunov Exponents). It gives information of whether nearby points are getting sucked closer together or ripped further apart as a system swooshes along at velocity $f(x_0)$.

There are things (not covered here) that can be said about stability along trajectories where the leading term is non-zero.
