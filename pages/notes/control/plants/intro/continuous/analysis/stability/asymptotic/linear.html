Linear systems

In a linear system, the Jacobian corresponds to the $A$ matrix.

If all of the eigenvalues of $A$ at an equilibrium point have a strictly negative real part, the system is stable at the equilibrium point.

If any of the eigenvalues are 0, the stability is inconclusive and fancier analysis tools would need to be used.
