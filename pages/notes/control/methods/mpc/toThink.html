Basics

In order to control a dynamical system, what are the steps to formulate it and solve it using MPC?

  • Formulate a least-squares cost function and minimise it.
  • The solution is expressed in a closed form as a solution of weighted least-squares problem.
  • The input constraints can be implicitly handled by properly selecting the weight matrices.

MPC algorithm usecase

  • Implement in python using classes.
  • Implement in C++ using the eigen library.
  • Test on a system consisting of two objects connected by springs and dampers.

Characteristics of systems considered

  • Unconstrained case of a linear dynamical system, where the state vector is completely known.
  • Case where the state is not known. An observer would need to be integrated with the MPC algorithm.
  • Constrained linear dynamical system
  • Non-linear smooth systems (the most general formulation)
