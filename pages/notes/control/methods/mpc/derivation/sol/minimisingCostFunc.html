MPC solution: Minimising the cost function derivative

Solution to the MPC problem

To minimise the cost function w.r.t. $u$, its derivative is set to $0$: $$\begin{align*} \frac{\partial{J}}{\partial{u}} &= -2M^T \, W_P \, s + 2M^T \, W_P \, Mu + 2Wu \\ &= 0 \\ (M^T \, W_P \, M + W)u &= M^T \, W_P \, s \end{align*}$$ $$ \hat{u} = (M^T \, W_P \, M + W)^{-1} \, M^T \, W_P \, s $$ $\hat{u} = \begin{bmatrix} \hat{u}_{k|k} \\ \hat{u}_{k+1|k} \\ \vdots \\ \hat{u}_{k+c-2|k} \\ \hat{u}_{k+c-1|k} \end{bmatrix} $ is the solution to the MPC problem.

Only the first control input is required. When $\hat{u}_{k|k}$ is applied to the system, after time instant $k+1$, state $x_{k+1}$ is observed, another cost function is formed and solution computed. $\hat{u}_{k+1|k}$ is then applied to the system and the whole procedure is repeated for the time instant $k+2$.
