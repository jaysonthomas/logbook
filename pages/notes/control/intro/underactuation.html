Underactuation

Definition

A quadrotor constrained to move in 2D has 3 DOF: $y,z,\phi$. However, it has only 2 control variables $u_1, u_2$ [Ref1, Ref2]. This means it is not possible to control all 3 DOFs simultaneously.

At any given time, a trajectory is defined to control for 2 DOFs, whilst sacrificing control of the 3rd. Such a system is called underactuated.

Coupling

$$\begin{align*} \ddot{z} &= g - \frac{u_1 \cdot cos\phi}{m} \\ \ddot{y} &= \frac{u_1 \cdot sin\phi}{m} \\ \ddot{\phi} &= \frac{u_2 \cdot Lcos\theta}{I_{xx}} \end{align*}$$ The $y,z$ behaviours are inherently coupled with roll. Any thrust will be shared between the 3 DOFs. If an attempt was made to simultaneously control $y, z, \phi$:
  • Assume $\phi$ is controlled first
    • It will require spending one control variable, in this case, $u_2$.

  • Next, controlling $y$ behaviour
    • Since $\phi$ is fixed, the other control variable is used to fix $y$, in this case, $u_1$.

  • Next, controlling $z$ behaviour
    • It is not possible to control for $z$, since all the control knobs available to influence the system has been used.
