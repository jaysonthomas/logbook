Differential equation

$$ \dot{x} = Ax + Bu $$ The equation describes how the state of a system changes over time due to both its current state $x$ and a control input $u$ applied to it. The equation can be used to model how a car's speed, for example, changes over time based on its current speed and the driver's input on the gas pedal.
  • $x$ - state of the system
    • Represents the speed of the car.

  • $u$ - control input
    • Represents the throttle position, indicating how much gas is being applied.

  • $A$ - System matrix
    • Represents how the current speed influences the next speed (for example, if there is no external force, $A$ might be 1, indicating that the speed remains constant if no gas is applied). Is it also sometimes called the state matrix?

  • $B$ - Input matrix
    • Represents how the throttle position affects the speed (for example, if pressing the gas pedal increases the speed by 0.1 m/s per unit of throttle position, $B$ might be 0.1).

Let's say the current speed of the car is 20 m/s, and the driver presses the gas pedal (when pressed down, $u=5$ i.e. the throttle is 5 units). Assuming, $A=1, B=0.1$, the new speed can be calculated as: $$ \dot{x} = 1 \times 20 + 0.1 \times 5 = 20.5m/s $$
