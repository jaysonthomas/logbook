TF: Example

Setup

Consider a MSD system on wheels.

The system model is a perfect candidate to convert into a transfer function, since:

  • It is an LTI system
  • There is a single input $F_{input}$ and a single output $x$

Finding the impulse response

To find the impulse response of the system, the single input is set to an impulse function, $F_{input}(t)=\delta(t)$ and solved for the response $x(t)$: $$ m\ddot{x}(t) + b\dot{x}(t) + k\overbrace{x(t)}^{\text{Impulse response} \atop \text{in the time domain}} - \underbrace{\delta(t)}_{\scriptsize\text{Impulse input force}} = 0 \\ $$ However, solving linear ODEs in the time domain can be time consuming. The task can be made easier by solving for the impulse response in the s domain, by taking the Laplace transform of the entire differential equation, one term at a time as shown here. The transfer function or the laplace transform of the impulse response of the system would be: $$ X(s) = \frac{1}{ms^2 + bs + k} $$

Intuition

Here, the transfer function represents the dynamics of the cart system and it can be used to predict how the cart will move when subjected to arbitrary forces, i.e. the impulse response is same as the cart dynamics.

Applying an impulse

An impulse can be applied to a transfer function by representing the input in the s domain and multiplying it with the transfer function. An impulse function in the s domain is 1 and therefore the impulse response is $1$ times the transfer function.

Step response

To find the step response of the system, a step input needs to be applied. To create a step input, an integrator is used.
An impulse input produces a step output through the integrator transfer function, which is then applied to the cart dynamics to produce the step response of the cart system.

The total transfer function for the system's response to the step input is the two transfer functions multiplied together. The definition of a transfer function is still retained, i.e. an impulse input is applied and an impulse response comes out - in this case the impulse response of the system is the step response.
