LTI: Properties

Overview

All LTI systems have the following properties:
  • Linearity
    • Homogeneity
    • Superposition
  • Time-invariance
That is, the inputs into an LTI system can be scaled, summed or shifted to still produce predictable outputs.

Linearity

Homogeneity

Scaling an input $x(t)$ by factor $a$ produces an output $y(t)$ scaled by the same factor. If a step input of height A produces an oscillating step to height B, then doubling the input will have the same affect on the output. $$ h(ax(t)) = a \cdot \underbrace{h(x(t))}_{y(t)} $$

Superposition (Additivity)

The response of a linear system to the sum of two separate inputs together, is the summed outputs of each individual input. If a step input A produces output $a$ and a ramp input B produces output $b$, then superposition states that if summing inputs A + B would result in the output $a + b$. $$ h(x_1 + x_2) = h(x_1) + h(x_2) $$

Combination

$$ h(ax_1(t) + bx_2(t)) = ay_1(t) + by_2(t) $$

Linear system vs linear equation

Linear system is a mapping between two vector spaces that obeys the properties of homogeneity and superposition. A linear equation is an algebraic expression where every term is a single, first-power variable multiplied by a constant.

A well-known example of a linear equation is the equation of a line in a 2D plane, $y = mx + b$. This equation is not homogenous because, for example, the output is not doubled when the input is doubled, $2(mx + b) \neq m(2x) + b$.

Time invariance

Time invariance refers to a system behaving the same regardless of when the action takes place, in time. If an input $x(t)$ is shifted by a fixed time $T$ then the output $y(t)$ is also shifted by that fixed time. $$ h(x(t - T )) = y(t - T ) $$
