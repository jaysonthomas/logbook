s Domain

Use

Instead of just cosine waves, the Laplace transform decomposes a time domain signal into both cosines and exponential functions.

Description

$s$ is a complex number, which means that it contains values for two dimensions:
  • One dimension describes the frequency of a cosine wave
  • The second dimension describes the exponential term
$$ s = \sigma + j\omega $$

s plane

Instead of two separate number lines to describe $s$ (one for frequency and one for the exponential rate), a 2D plane is used where the real axis is the exponential line and the imaginary axis is the frequency line. The value of $s$ provides a location in this plane, describing the resulting signal, $e^{st}$, as a function of the selected $\omega$ and $\sigma$ [Ref].
