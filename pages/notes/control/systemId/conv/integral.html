Convolution Integral

Motivation

How is a system's impulse response useful in predicting its behaviour to arbitrary inputs, since infinitely high and infinitesimally thin input forces are physically impossible to create.

Convolution gives the ability to determine an LTI system's response to any arbitrary input as long as the impulse response of the system is known.

Definition

Convolution integral is a mathematical operation typically performed using 2 functions: $$ (f*g)(t) = \int^\infty_{-\infty}f(\tau)g(t-\tau)d\tau $$ The equation:
  • Shifts the system's response $g(t)$ through all of time
  • Multiplies the shifted $g(t)$ by the input $f(t)$
  • Sums the product over all of time

Time shift notation

Analogy

Imagine a race where the starting line is the origin. If runner A starts at the starting line ($t = 0$), and runner B starts $T$ seconds later, then at any given time $t$, runner B will have been running for $t - T$ seconds.

Context

In the context of an impulse function, the independent variable $t$ represents the time elapsed since the origin or reference point. $\delta(t)$ is defined as an impulse occurring at $t = 0$.

If a new impulse occurs $T$ seconds after another, to make it the new reference, a new representation would need to be used. In the new representation, the time at $t=T$ is represented as $t'=0$ by shifting the time frame for all signals, $t' = t - T$. When $t=0$, $-T$ seconds would have elapsed since the new reference, i.e. $t=0$ happened before $t=T$. At $t=T$, 0 seconds have elapsed w.r.t. the reference time $T$.

Integral limits in the real world

In practical applications, signals start at time zero. Since there is no signal between negative infinity and zero, there is no need to perform integration over the region. Whereas a pure mathematical function does go from negative to positive infinity.

Example: Multiplication of 2 polynomials

Polynomial multiplication involves multiplying every term of 1 polynomial with every term of the other and summing like units.

Consider $x^2 + 2x + 3$ and $3x^2 + 1$. The answer is $3x^4 + 6x^3 + 10x^2 + 2x + 3$.

If the 2 polynomials are defined by the vectors $f(t) = [1,2,3]$ and $g(t) = [3,0,1]$, the discrete convolution would sweep $g(t)$ across $f(t)$ and sum the product of the two.
