Question

What update needs to be made to $U$ in the next iteration such that the error across the entire run is reduced.

Steps

  • The error is filtered through a learning function $L$ to produce a $\Delta U$ signal, which is added to $U_k$ to get the new input $U_{k+1}$.
  • Often a low pass filter $Q$ is added here to remove high-frequency noise and to reduce chattering in the controller.
  • The updated $U$ is input into the system to produce $Y_{k + 1}$, which is compared to the same reference as before to get a new error for the iteration.
  • Repeat the cycle.
