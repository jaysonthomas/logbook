Graident Descent

Overview

Create a cost function $J$ that's a function of the error squared, where the error is just a function of the input $U$: $$\begin{align*} \min_u \quad & J_k(u) = \frac{1}{2} E_k^T E_k\\ s.t. \quad & E_k = R - GU_k \end{align*}$$ To lower the error between iterations, input $U$ needs to be changed in the direction that lowers $J$. This is done by taking a partial derivative of J w.r.t. $U$ to get the gradient $\frac{\partial J_k}{\partial U_k}$.

$k$ is then adjusted in the negative direction or in the direction that descends the gradient. $$ U_{k+1} = U_k - \frac{\partial J_k}{\partial U_k} $$
