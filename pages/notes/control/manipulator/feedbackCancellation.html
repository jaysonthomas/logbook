Feedback cancelling control

Description

[Example]
Given the standard manipulator equation of motion, if $M, C, \tau_g$ are known, then a controller that changes or overwrites a system's effective dynamics can be written if the change is:
  • Local, within reason
  • Small
This idea of cancelling the natural dynamics of a system is called feedback cancelling control. For example, a 2 link pendulum can be made to defy gravity. This is not always possible since unlimited control authority is seldom available.

Limitations

  • This works only in the case of a known system
    • This method is dependent on being able to estimate the torques, masses, coriolis and gravitational terms accurately. People do this! A sensitivity analysis can be done to check how sensitive the controller is to the mass matrix estimate and the like.

  • A manipulation system is an underactuated system
    • For the dynamics of a full manipulation problem which include both the masses of the robot and the manipuland, the equations of motion cannot be written with just $u$ anymore because direct torque cannot be applied to objects in a robot's environment; the torques are applied to the robot joints. Therefore, dynamics of the objects in the world cannot be directly overwritten. $$M(q)\ddot{q} + C(q,\dot{q}) = \tau_g(q) + Bu$$ A $B$ matrix would have to be introduced to indicate to what joints/actuators/objects, torque can be directly applied. $B$ would be low rank since torque cannot be applied directly to objects in the environment. For this reason a manipulation system (which includes both the world and the robot) is an underactuated system.

  • Limitations in the robot
    • To follow any arbitrary dynamics, may require an unreasonable amount of torque which may be beyond a robot's torque limits. These limits would make a system underactuated.
