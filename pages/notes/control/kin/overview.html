Overview

The single task problem can be solved by computing a Jacobian pseudo inverse. But, this solution has issues when it encounters Singularities or joint limits.

A quadratic program solution to the problem of diff IK for a single task will help cope much more easily with these problems. This is easy to implement if an IK solver or a controller that uses this is required - it involves plugging the QP equation into an off-the-shelf QP solver.
