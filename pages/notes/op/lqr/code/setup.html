Setup

Overview

The aim is to simulate a discrete-time LQR controlling a simple 1D motion system (position + velocity), and publish the trajectory over time. Refer double integrator.

The control input $u$ is acceleration.

This is an LQR stabilisation problem.

State

The system starts displaced from the origin. The controller's job is to drive the state to 0. $$x_0 = \begin{bmatrix} 1 \\ 0 \end{bmatrix}$$

Control input

The optimal feedback gain $K$ is computed using LQR. The optimal control law is: $$\begin{align*} u_k &= -K \; x_k \\ &= - \left(k_1 p_k + k_2 v_k \right) \\\\ K &= \begin{bmatrix} k_1 & k_2 \end{bmatrix} \end{align*}$$ Since the aim is to drive the state to 0, any value of position or velocity is an error and is used to determine the acceleration. If position is positive, controller applies negative acceleration to bring it back.

Cost matrices

LQR minimises the cost: $$ J = \sum_{k=0}^\infty x_k^\top Q_k x_k + u_k^\top R_k u_k $$ $$ Q = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} $$ $$ R = 0.1 $$

Interpretation

  • Large state values are penalised.
  • Since control effort is not penalised as much, the controller can act aggressively.

Simulation parameters

$$\begin{array}{ccl} h = 0.1 &:& \text{Time step} \\ t_f = 10 &:& \text{Total simulation time} \\ N = \frac{t_f}{h} + 1 &:& \text{Number of simulation steps} \end{array}$$
