Overview

The aim is to minimise a cost function $J$ w.r.t. state $x(t)$ and control $u(t)$ trajectories. $J$ is a function of functions since the state and control are functions of time. $$ \min_{\substack{x(t) \\ u(t)}} \quad J(x(t), u(t)) = \int_{t_0}^{t_f} l(x(t), u(t)) \; dt + l_f(x(t_f)) $$
$$\begin{array}{ccl} l(x(t), u(t)) &:& \text{Stage cost. Cost incurred at each time step.} \\ && \text{This is added along the trajectory.} \\ l_f(x(t_f)) &:& \text{Terminal cost that is just a function of the final state (the final landing spot)} \\ && \text{since there are no more moves to perform.} \end{array}$$
This is a finite time problem where once the end is reached at the final time step, there are no more control moves and a final cost is incurred.

The solutions are open loop trajectories in $(x, u)$. In a stochastic setting, the solutions are feedback policies since everything is noisy and some feedback is needed to stabilise everything. RL can only provide feedback policies as solutions.

Aside from LQR, there are no other problems of this nature that have analytical solutions. That is where numerical methods/optimisations are used.

Constrained optimisation problem

$$ s.t. \quad \dot{x}(t) = f(x(t), u(t)) $$ The constraint is a dynamics constraint. It is a set of equality constraints that are enforced at every time step.

Other constraints:

  • Actuator constraints: bounds on $u$ which are inequality constraints.
  • Collision avoidance constraints: which are state inequality constraints.
