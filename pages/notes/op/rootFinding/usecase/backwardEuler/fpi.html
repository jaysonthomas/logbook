Fixed point iteration

Aim

Refer Euclidean norm.

The aim is to compute the next state $x_{k+1}$ given the current state and the system dynamics. Computation of the next state at each step is an iterative process.

Euler integration is based on approximating the next discrete state by linearising the continuous dynamics using Taylor series expansion.

Iterative loop

Consider that the code is executing the very first time step ($k=1$). Hence, the aim is to compute $x_1$. $$\begin{align*} \tilde{x}_0 &= x_0 \\ \tilde{x}_1 &= x_0 + h f(\tilde{x}_0) \\ \tilde{x}_2 &= x_0 + h f(\tilde{x}_1) \\ \vdots &= \vdots \end{align*}$$ This can be generalised as: $$\begin{align} \tilde{x}_{k+0} &= x_k \nonumber \\ \tilde{x}_{k+1} &= x_k + h f(\tilde{x}_{k+0}) \nonumber \\ \tilde{x}_{k+2} &= x_k + h f(\tilde{x}_{k+1}) \nonumber \\ \vdots &= \vdots \nonumber \\ \tilde{x}_{k+n+1} &= x_k + h f(\tilde{x}_{k+n}) \label{eq:1} \end{align}$$
$$\begin{array}{ccl} k &:& \text{Current time step} \\ n = [0, 1, \cdots] &:& \text{Number of iterations within the current time step} \\ && \text{to find the next state as accurately as possible.} \\ \tilde{x}_{k+n+1} &:& \text{Output state from the }n^{th} \text{ iteration.} \\ f(\tilde{x}_{k+n}) &:& \text{Jacobian of the dynamics.} \end{array}$$

Stopping criteria

The iteration stops when the following criteria is met: $$\begin{align*} e_k &\leq \epsilon \\ e_k &= \|{\Delta{\tilde{x}}_k}\|_2 \\ \Delta{\tilde{x}}_k &= \tilde{x}_{k+n+1} - \tilde{x}_{k+n} \end{align*}$$

Intuition

\ref{eq:1} is of the form $$ y = c + xm $$
$$\begin{array}{ccl} m &:& \text{Jacobian} \\ x = h &:& \text{Size of the time step which is a constant.} \\ c = x_k &:& \text{The y-intercept is the current state} \\ && \text{which is the initial state for the iterative loop.} \\ y = \tilde{x}_{k+n+1} &:& \text{The next state} \end{array}$$
The output of the first step is the same as the output of forward euler integration, where the tangent line always takes the next discrete state slightly away from the actual dynamics function curve. With further iteration, the slope changes, bringing the next discrete state closer to the actual next continuous state.

At every state, it is assumed that the computation of the next state will converge to a stable fixed state. At a fixed point, the derivative is $0$, i.e. the next state will be the same as the current state in the iterative process. Hence why the stopping criteria checks the difference between states.

The fixed point at every iterative step is not the same as an equilibrium point for the overall dynamics of the system. Locally, every state (globally stable or not) is assumed to want to converge to a particular stable next state.
