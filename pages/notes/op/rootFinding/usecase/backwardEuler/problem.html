Problem description

Overview

To simulate a simple pendulum, a secondary aim is to compute the next state $x_{k+1}$ given the current state and the system dynamics. Computation of the next state at each step is an iterative process. Backward Euler math.

X-Y

The objective function is the residual: $$ f(x_{k+1}) = r(x_{k+1}) $$ At each step, for a particular value of $x_{k+1}$ (X-axis), the residual (Y-axis) becomes ~0.

Stopping condition

Euclidean norm of the residual is approximately 0.
