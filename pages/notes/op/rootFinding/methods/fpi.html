Fixed point iteration

Setup

Consider a dynamical system that is stable (all eigenvalues have negative real parts if continuous time or are within the unit circle in a complex plane if discrete time) at an equilibrium point that is unknown.

Description

If the system is iterated forward in discrete time or simulated forward in continuous time, the system will eventually land at the stable fixed point or zero.

Only works if the equilibrium point to be found $x^*$ is stable and if the initial guess is in the basin of attraction. The method can converge slowly depending on the properties of $f$ (the magnitude of the eigen values). An asymptotic convergence means it will get to the zero eventually, but could take a long time.

Usecase

Gradient descent is fixed point iteration (the optimisation algorithm) applied to minimising the gradient of a function as the objective function.

Consider a scalar function $f$. Solving for gradient of the function equal to 0 to find the root. The gradient can be seen as the equivalent of the 'dynamics' in this case.

All the disadvantages of FPI applies to gradient descent. That is the reason why there are so many tricks being applied to make gradient descent work.

Performance

This is O($n$).
