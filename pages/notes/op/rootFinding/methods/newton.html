Newton's method

Steps

[Oscar Veliz, First order taylor expansion]

Fit a linear approximation to $f(x)$ (there is always an initial guess) $$ f(x + \Delta{x}) \approx f(x) + \frac{\partial{f}}{\partial{x}}\bigg|_x \, \Delta{x} $$ At $x_k$, the taylor approximation of the function would be the following, where $\Delta x = x_{k+1} - x_k$. $$ f(x_k + \Delta{x}) \approx f(x_k) + \frac{\partial{f}}{\partial{x}}\bigg|_{x_k} \, \Delta{x} $$ It is hard to solve a nonlinear equation $f() = 0$. But, solving a linear approximation is easy. Hence, set approximation to 0 and solve for $\Delta{x}$ $$\begin{align*} 0 &= f(x) + \frac{\partial{f}}{\partial{x}}\bigg|_x \, \Delta{x} \\ \Delta{x} &= -\left(\frac{\partial{f}}{\partial{x}}\bigg|_x\right)^{-1} \, f(x) \end{align*}$$ Apply the correction. $x + \Delta{x}$ becomes the new $x$. $$ x + \Delta{x} \rightarrow x $$ Repeat until convergence.

Performance

Newton's method has quadratic convergence. It's second order accurate. The algorithm zeroes out the error in the first order taylor expansion at each iteration. The only error left starts at the second order term in the taylor expansion.

It is necessary for the initial guess to be in the basin of attraction for the method to converge at all.

Most ML algorithms use fixed point iteration or gradient descent. This is because Newton's method has to solve a linear system by inverting the hessien matrix which is expensive in general for large problems. It is O($n^3$) where $n$ is the size of the vector $x$. See julia inversion operation.

The solution is to use quasi-newton methods like BFGS which brings down the complexity to O($n^2$). Another key method is to exploit problem structure - often, problems that have big dense matrices are sparse (with a lot of 0s) and have a lot of structure under the hood, i.e. patterns in how the non-zeros show up.

It is possible to exploit that sparsity and that structure of which blocks are nonzero to do this way cheaper. For control problems, it is possible to bring the complexity down to almost linear time ~O($n$) in the time Horizon of the control problem.
