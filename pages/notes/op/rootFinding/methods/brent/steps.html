Brent's method: Steps

Overview

Brent's method is essentially a careful orchestration of other root finding methods with the aim of guaranteeing that the best root will be found in the shortest amount of time.

Convergence check

If $|b-a| \leq \epsilon$ or $f(b)==0$, $b$ is the root.

Ensure $b$ corresponds to the best solution

If $|f(a)| < |f(b)|$ $$ a \leftarrow b, b \leftarrow a, c $$

Compute new candidate abscissas

If $f(a), f(b), f(c)$ are distinct values, IQI is chosen. Else, secant method is chosen to find a candidate $s$.

$m$ is the midpoint of the range found using bisection method.

Compute new boundary

$$\begin{align*} k &= \frac{3a + b}{4} \\ &= 0.75a + 0.25b \end{align*}$$

Check secant or IQI point

If $s$ lies between $k, b$.
  • $f(s)$ does not have the same sign as $f(a)
    • $$ a, b \leftarrow s, c \leftarrow b $$
  • $f(s)$ has the same sign as $f(a)
    • $$ a \leftarrow b, b \leftarrow s, c \leftarrow b $$

Check midpoint

If $f(m)$ does not have the same sign as $f(a)$: $$ a, b \leftarrow m, c \leftarrow b $$ Else $$ a \leftarrow b, b \leftarrow m, c \leftarrow b $$
