Line search

Overview

Often $\Delta{x}$ step from Newton overshoots the minimum.

To fix this, check if the objective function at the new iterate $f(x + \Delta{x})$ went up or down versus $f(x)$. The aim is for it to go down. Back track or shrink the size of $\Delta{x}$ until a good reduction in $f$ is obtained. The definition of good is where all the variations of this algorithm differ.

Simply checking for a decrease won't work as it can easily lead to an overshoot.

Armijo rule

Start with some $\alpha$ that is going to be the step length. $$\begin{align*} &\alpha = 1 \\ &\text{while } f(x + \alpha \, \Delta{x}) > f(x) + b \, \alpha \, \nabla{f(x)}^\top \Delta{x} \\ &\qquad \alpha \leftarrow c \, \alpha \\ &\text{end} \end{align*}$$
$$\begin{array}{ccl} \alpha \, \nabla{f(x)}^\top \Delta{x} &:& \text{Expected reduction in } f \text{ based on a first order taylor expansion} \\ b &:& \text{User defined tolerance. Usually }10^{-4} \text{ to } 0.1 \\ c &:& \text{Scalar < 1. Helps decrease the step size. Usually }0.5 \\ \end{array}$$
With $b$ added on the RHS of the while check, it checks whether the actual decrease in $f$ agrees with a first order taylor expansion to within a tolerance dictated by $b$.

Intuition

A first order Taylor expansion or linearisation is used to compute the $\Delta{x}$ step.

To trust the step, the output of the actual nonlinear function needs to be less than an even smaller step decided by the tolerance factor taken by the taylor expansion.

Armijo just uses a first order taylor expanasion in its check. The fancier Wolf conditions use a second order taylor expansion. Even fancier line searchers called cubic line search use a third order.

$b, c$ are heuristics. If a lot of time is being spent on tuning these parameters, then more sophisticated methods need to be used. Fancier line searches such as filter line search (a 2D search that considers both constraints and the objective) used by ipopt.

Performance

Easily parallelisable. Solutions can be checked for many $\alpha$s in parallel and the best one out of these can be picked.

Regularisation with backtracking guarantees a monotonic decrease to the local minimum.
