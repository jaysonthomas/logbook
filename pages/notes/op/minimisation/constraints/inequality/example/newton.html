Newton step

Main call

newton_solve is given a warm-start with a $\rho = 1$ initially. This relaxes the complementarity condition to equal 1.

With every new point $z$, a new value of $\rho$ from $1$ to $1e^{-8}$ is used in the call to newton_solve which drives the solution closer to the constraint along a path called the central path. 


    function newton_solve(z0,ρ,tol)
There are better ways to adjust $\rho$. One method is to monitor the constraint violation and residuals and scale $\rho$ based on how close the solution is to desired solution.

If the starting point is in the interior and $\rho$ is small, there won't be much contribution from the gradients of the constraints to push the solution in the right direction. But, as a solution gets closer to the constraint, $\rho$ can be cranked down. Would it lead to ill conditioning if $\rho$ was high when a solution is close to the constraint? The constraint curve is equivalent to a brick wall.

Before the while loop


    #initial guess
    z = z0
    
    #KKT residual
    r = ip_residual(z, ρ)

While loop condition

The norm of the residual needs to be as close to 0 as possible. Hence, the while condition becomes: $$ \|r\|_2 > \epsilon $$

Inside the loop

Compute new point using the Jacobian

Compute the new point $z_n$: $$\begin{align*} \Delta z &= -J^{-1} \; r \\ z_n &= z + \Delta z \end{align*}$$ Compute the residual for the new point: 

      rnew = ip_residual(znew, ρ)

Line search


      b = 0.1
      c = 0.5
      α = 1.0
      while norm(rnew) > (norm(r) + b*α*dot(r,J*Δz)/norm(r))
          α = c*α
          znew = z + α*Δz
          rnew = ip_residual(znew,ρ)
      end

      z = znew
      r = rnew
