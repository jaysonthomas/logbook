Landscape plot

Objective function

$x$ is a 2 \times 1 vector in 2 dimensions with a horizontal-axis (x) and a vertical-axis (y) value. Feed the function $f$ with different (x, y) values to get a scalar which acts as the altitude in the contour plot.

Constraint

Only the boundary where the constraint is satisfied at $c(x) = 0$, is plotted.

$c(x)$ defines a relationship between $x_1$ and $x_2$. Setting the function to 0 to find the boundary: $$ x_1^2 + 2x_1 - x_2 = 0 $$

To plot this in a standard 2D coordinate system (where $x_1$ is the horizontal axis and $x_2$ is the vertical axis), solve for $x_2$: $$ x_2 = x_1^2 + 2x_1 $$ This is the standard equation for a parabola, which is a 1D line living in a 2D space.
