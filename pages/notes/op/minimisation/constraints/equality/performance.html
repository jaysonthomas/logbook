Performance

It is not necessary to give the root finding problem a feasible initial guess.

Equality constraints are usually going to be the dynamics of a robot which are non-linear. Giving a feasible initial guess would be hard.

To avoid overshooting, line search would need to be implemented. A linearisation of the constraint (which can be a non-linear function) is used at each step. So, each solution would have to lie on that approximated tangent constraint-line - it moves towards the minima at each step, but lies on the approximated tangent constraint-line. It is not guaranteed to stay on the actual constraint curve. But, it will track it well; it will eventually move towards it.

At each step/spot, $\lambda$ tries to impedance match between $\nabla f$ and $\nabla c$ so they cancel each other out. It will definitely be a non-zero value.

In the example, the Hessian of the objective is positive definite. But, the Hessian of the Lagrangian which includes the curvature of the constraint can be indefinite based on the starting location. This can cause the solution to get stuck in a local optimum that is not even on the constraint. In the example, the objective function is convex, but the constraint isn't and the constraint curvature term can mess things up. Regularisation is the right solution.

Regularisation involves messing with the constraint curvature term which mutes its usefulness. It would be useful if the initial guess is closer to the optimum. Regularisation can make things worse if the initial guess is far from the optimum. It might be better to just ignore the term as in the Gauss-Newton method. An indefinite Hessian is bad because it corresponds to something non-convex.

In control problems, the user gets to design the cost function $f$ which is usually a benign convex or quadratic function. However, the user does not get to pick the constraints which usually involves the physics/dynamics, safety etc. The constraints tend to be the tricky function. Hence why Gauss-Newton is often used since the Hessian of the objective function behaves well towards the Hessian of the Lagrangian.
