Zero gradient

Overview

The gradient $\nabla{f} = 0$ in the directions the system is free to move in.

The system can only move in certain directions/dimensions. The gradient can be non-zero in the non-free directions.

For the unconstrained version, $\nabla{f} = 0$ is the only FON condition for a minimum [Ref].

Math

Mathematically, the necessary condition on the gradient is described as below. See intuition.

Scalar case

Any non-zero component of $\nabla f$ must be normal to the constraint at an optimum. Equivalently, $\nabla f$ must be parallel to $\nabla{c}$. There is no need for the gradients to be of the same length. $$ \nabla{f} + \lambda \, \nabla{c} = 0 $$ $\lambda$ is the Lagrange multiplier or dual variable.

Higher dimensional space

$$ \frac{\partial{f}}{\partial{x}} + \lambda^\top \frac{\partial{c}}{\partial{x}} = 0 \qquad \lambda \in \mathbb{R}^m $$ $c$ holds a bunch of scalar constraints, one for each dimension. There is a $\lambda$ for each one of those constraints.
