Root finding: Description

Overview

The root of a function is the value of $x$ (abscissa) for which $f(x) = 0$. The scalar output of the function can also be referred to as the ordinate. In some cases, it is sufficient for the output to be approximately 0.

Overview

[Ref]

Root finding refers to the general problem of searching for a solution of an equation $F(x) = 0$ for some function $F(x)$.

There are few examples where there exist exact methods for finding solutions. For example, the quadratic formula: $$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$ gives an exact method for finding roots of the equation $$ ax^2 + bx + c = 0 $$
