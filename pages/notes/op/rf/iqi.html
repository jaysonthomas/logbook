Root finding: Inverse quadratic interpolation

Initialisation

Obtain a set of three points $x,y$ where $x$ is the abscissa value and $y$ is the value of the objective function evaluated at $x$, i.e. $y=f(x)$.

Iteration

  • Compute a new abscissa value based on IQI
    • $$ x_{new} = a \frac{f_b}{f_a - f_b} \frac{f_c}{f_a - f_c} + b \frac{f_a}{f_b - f_a} \frac{f_c}{f_b - f_c} + c \frac{f_a}{f_c - f_a} \frac{f_b}{f_c - f_b} $$

  • Evaluate the new abscissa
    • $$ y_{new} = f(x_{new}) $$

  • Update the 3 points by throwing away the oldest
    • $$\begin{align*} (x_a, y_a) &\leftarrow (x_b, y_b) \\ (x_b, y_b) &\leftarrow (x_c, y_c) \\ (x_c, y_c) &\leftarrow (x_{new}, y_{new}) \\ \end{align*}$$

  • Evaluate some appropriate stop criteria
    • Keep iterating until the stop criteria is met, in which case the value of $y_{new}$ should be approximately $0$. It is advised that a “best solution” is maintained, as

  • Buffer the best solution
    • Since $x_{new}$ is not guaranteed to be better than previous candidates, a copy of the best solution is maintained.
