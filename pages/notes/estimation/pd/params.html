Description Estimating parameters from data

PD: Parameters

Description

$\theta$ is used to collectively refer to parameters that describe a distribution. For example, a continuous distribution like a Gaussian, can be fully described by two parameters $\mu$ and $\sigma^2$.

$p$ of $x$ given $\theta$ is not the same as the function that describes the density of $x$, parameterized by $\theta$. $$p(x|\theta) \neq p(x;\theta)$$ $p(x|\theta)$ implies $\theta$ is a random variable itself.

Estimating parameters from data

Consider samples $x$ are being generated from some Gaussian $\mathcal{N}(\mu, \sigma^2)$ with probability density $p(x)$ and whose parameters $\mu$ and $\sigma$ are unknown. The estimates of these parameters are written as $\hat{x}$ and $\hat\sigma^2$ respectively.
If samples of data $\{x_1, x_2,.., x_n\}$ are repeatedly drawn from the distribution, more data will appear near the mean, and fewer samples out on the tails of the gaussian. Using a technique called the method of moments, it can be shown that: $$\hat{x} = \frac{1}{n} \sum^n_{i=1}x_i$$ $$\hat{\sigma}^2 = \frac{1}{n} \sum_{i=1}^n (x_i-\hat{x})^2$$
