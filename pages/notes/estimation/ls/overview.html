Least Squares Overview

Overall idea of LS

Carl Friedrich Gauss' description of Least Squares:
The most probable value of an unknown parameter is that which minimizes the sum of squared errors between the observed and the expected, multiplied by numbers that measure the degree of precision (weights or uncertainties associated with each observation).

The least squares method aims to fit a mathematical model to a set of observed data points. Each data point has an associated uncertainty or error (weight), which represents how much that specific observation can deviate from the true value.

Gauss was suggesting that to find the most probable value of an unknown parameter, one should consider
  • The squared differences between the model's predictions and the observed data points
  • The uncertainties associated with each data point
Multiplying the squared error by weights gives more importance to data points with lower uncertainty. This way the overall error is getting minimised whilst also taking into account the quality or reliability of each data point.

A weight can be thought of as the inverse of the variance (or standard deviation) associated with each data point. Data points with smaller variances (higher precision) contribute more to the parameter estimation in the least squares method.

Getting the model right is hard!

Relation with Bayes filters

To estimate a state that is changing, the Kalman filter and its variants called extended and unscented kalman filters can be used. For example, to estimate the position of a vehicle, a kalman filter helps incorporate the vehicle's motion into the state estimation process.

Else, the least squares approach would suffice.

Assumptions in the basic LS approach

The most basic least squares approach would make unrealistic assumptions in the following areas:
  • Sampling time
    • Measurements come in all at once.

  • State of the system at the time of measurement
    • The state or parameter being estimated is constant during and between measurements. Eg: a vehicle is stationary.

  • Measurement model
    • The underlying measurement model $H$ is noisy but linear. Measurements of things like orientation are rarely linear.

  • Accuracy of measurements
    • Each measurement has an equal weight in the error equation.
