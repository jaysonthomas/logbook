Use

Most often systems are partially observable and very noisy which means it is not possible to reconstruct a full, clean state from just current measurements.

Consider only position measurements are available. A state could include both position and velocity. Velocity cannot be computed if only the current position measurement is known.

A naive way to compute velocity would be to finite difference the current position and the last position. But finite differencing amplifies noise. By using a kalman filter and leveraging the dynamics model, a much better job can be done.
