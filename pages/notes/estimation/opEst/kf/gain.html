Kalman Gain

Using the MMSE idea of minimising the state covariance which is the same as minimising the trace of the updated covariance $\Sigma_{k+1|k+1}$: $$\begin{align*} \argmin \quad &E[(x_{k+1} - \hat{x}_{k+1|k+1})^T(\dots)] \\ &\text{tr}(\Sigma_{k+1|k+1}) \end{align*}$$ Using the covariance equation, minimise the trace of it to find the optimal $L$ by setting the derviative of the trace to 0 and solving for $L_{k+1}$. $$\begin{align*} \frac{\partial \text{tr}(\Sigma_{k+1|k+1})}{\partial L_{k+1}} &= 0 \\ L_{k+1} &= \Sigma_{k+1|k} C^\top S_{k+1}^{-1} \end{align*}$$
