Setup

Assumptions

Consider a recursive linear MMSE estimator. Recursive means a state estimate is maintained/updated at each time step.

State estimate

The state estimate at time $k$ is assumed to include all measurements up to the current time. $$ \hat{x}_{k|k} = E\left[x_k | y_{1:k}\right] $$
$$\begin{array}{ccl} \hat{x}_{k|k} &:& \text{Estimate at k given or conditioned on k} \\ E\left[x_k | y_{1:k}\right] &:& \text{Expectation of } x_k \text{ conditioned on measurements from time step 1 up to } k \end{array}$$
The current estimate is causal as in it only depends on current and previous times, not on future data.

State covariance

The state covariance, error covariance of the state, is assumed to be known. In other words, gaussian beliefs/distribution over the state is assumed. $$ \Sigma_{k|k} = E\left[(x_k - \hat{x}_{k|k}) (x_k - \hat{x}_{k|k})^\top\right] $$

Aim

The aim is to update $\hat x$ and $\Sigma$ to include a new measurement at $t_{k+1}$.
