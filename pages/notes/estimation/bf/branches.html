Bayes Filters: Branches

  • Kalman Filter
    • Kalman filter is a specific implementation of the bayes filter where everything is represented as Gaussians.

  • Extended Kalman Filter (EKF)
    • Here, everything is Gaussian but the state or observation models are non-linear. The Jacobian matrix (given by the models) is linearised around the current estimated state to make things Gaussian again. Everything then works out just like a Kalman Filter. Linear approximations have to be made at every time step. EKF is hard to implement because even the smallest amount of error throws the estimator.

    In both KF and EKF, the beliefs are described by Gaussians.

  • Unscented Kalman Filter (UKF)
    • Here, a non-linear state is estimated using special particles which can be passed through a non-linear system instead of linearising the system as in the case of EKF. It is simpler to implement.

  • In Particle Filter, particles or samples are used to estimate a non-linear distribution which is not Gaussian.
