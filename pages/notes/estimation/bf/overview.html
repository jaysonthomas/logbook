Bayes Filters: Overview

Novelty

  • Bayes Filter is a general framework for thinking about state estimation. It updates and maintains a belief of a state that is continuously changing.

  • Bayes filter incorporates measurements as well as control into the state estimator in continuous space.

    • It is not possible to maintain a robust and responsive estimate of a state (eg. position of a moving vehicle) with noisy sensor data, unless the estimator is also made aware of the controls the system is commanded.

Key steps

Bayes filters alternate between 2 steps:
  • Prediction step
    • A future state can be predicted using the control actions commanded of the system and the system's dynamics; in other words, using the predicted effects of the control actions.

  • Measurement update step
    • Observations can be used to correct the prediction.

Uncertainty in the estimate after each step

The uncertainty in a state estimate is not a point estimate but a distrbution.
  • After the prediction step
    • The current state is projected forward based on the control input. This prediction will be noisy because of imperfections in the system model, causing an increase in the uncertainty and hence the belief to spread out.

    If a transition model is perfect with no noise, it would be possible to predict forever without the need to make any measurements. However, because there is noise, if the prediction step is carried out forever, the transition noise would keep increasing after each step.

  • After the measurement step
    • The uncertainty goes down even though the measured information from the external world is noisy.
