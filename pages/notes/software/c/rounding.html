lroundf

C: Rounding

lroundf

[Ref]
Rounds halfway cases away from 0. Examples given in the reference. signbit returns true for negative values and the first part of the ternary operation gets executed.
  • $0.2 = 0.0$
  • $0.4 = 0.0$
  • $0.5 = 1.0$
  • $0.7 = 1.0$
  • $-0.2 = 0.0$
  • $-0.5 = 1.0$
  • $-0.7 = 1.0$
