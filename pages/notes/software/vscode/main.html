C++ include path

VS Code's IntelliSense engine is separate from the actual compiler (like GCC or Clang). Even if the code builds fine using a Makefile or CMake, the editor needs to be "told" where to look so it can provide helpful suggestions.

Steps:
  • Ctrl + Shift + P
  • Type C/C++: Edit Configurations (UI)
  • Scroll down to the Include path section
  • Include /usr/include/eigen3/** or /usr/local/include/eigen3/**
  • Alternatively, click the "JSON" link or open .vscode/c_cpp_properties.json and update the includePath array

Editor tweaks
