Bazel concept: External repositories

Overview

In Bazel, @something refers to an external repository. External repositories are dependencies that are fetched and made available in a Bazel workspace. They can be git repositories, or archives (zip files) or local file system directories.

Defining an external repository

Instead of using http_archive or git_repository in WORKSPACE, dependencies are now defined in MODULE.bazel using bazel_dep or use_repo.

If @something is not resolved in MODULE.bazel, the module may be missing from the Bazel central registry.
