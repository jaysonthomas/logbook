Bazel concepts: Module

Overview

[Ref]

A module is the root folder or package that contains all the data that is needed to build a project. It is identified by a MODULE.bazel file. It is represented by // in a path. It used to be referred to as workspace instead.

Each folder of a module can act as a package (including the root directory) and identified by a BUILD or BUILD.bazel file.

Contents of MODULE.bazel

  • Declaration of the name of the module.
  • Minimum bazel version required.
