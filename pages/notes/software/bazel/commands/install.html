Bazel commands: Installation

Bazilisk

Bazilisk acts as a version manager for Bazel. It automatically downloads and runs the latest compatible version defined in:
  • A .bazelversion file in a project.
  • The USE_BAZEL_VERSION environment variable.
  • The latest stable version (if no version is specified).

Installation

Fetch the latest Bazelisk binary for Linux (x86_64) 


// Fetches to /usr/local/bin/bazel
sudo curl -Lo /usr/local/bin/bazel https://github.com/bazelbuild/bazelisk/releases/latest/download/bazelisk-linux-amd64

// Make the binary an executable.
sudo chmod +x bazelisk-linux-amd64

Check if Bazelisk is correctly installed. The first run will download the appropriate Bazel version (default or as specified in a project). 


bazel --version

Uninstall


sudo rm /usr/local/bin/bazel
