Setup

URDF

Add control tags in the URDF and define the hardware interface which is a mock initially.

Specify the command and state interfaces which should match that of a controller specified in the YAML file.

YAML

Create yaml file under bringup/config to configure the controller manager which will start the different controllers and bind them with hardware interfaces. The latter are described in a urdf file.

controller manager parameters include update rate, joint state broadcaster and controllers to be used. Nodes specified as parameters will be started by the controller manager.

Each controller is loaded in as a plugin. To find out what type to specify in the YAML file, check the class name specified in the plugin.xml file of the corresponding controller to be used. Parameters that can be used with the controller can be found in a YAML file in the src folder of the controller.

Specify interface type which should match that of the hardware interface specified in the URDF.
