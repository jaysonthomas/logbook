ros2_control is a framework for (real-time) control of robots using ROS 2. It provides:
  • A hardware abstraction layer - abstractions on hardware interfaces
  • Controllers that consist of a feedback mechanism, can receive a setpoint and the control an output that is applied to a real robot or to a simulation in Gazebo for which a Gazebo plugin adapter would be required.
