Sim from scratch

Steps overview

  • Create ros workspace/src, description package.
  • Connect URDF, Rviz and launch file. Build.

  • Configure ros2_control
    • Create hardware interface tag in urdf.

Setup


    sudo apt install ros-kilted-ros2-control ros-kilted-ros2-controllers
    ros2 pkg create --build-type ament_cmake --license Apache-2.0 description
    colcon build
    ros2 launch description display.launch.py
Create URDF file with the various joints/links attached to the world. Pass this information to the robot state publisher which takes care of computing the kinematics. Link rviz, joint state publisher gui and robot state publisher nodes.

robot state publisher listens for joint states and publishes them on the TF.

joint state publisher gui publishes fake joint states. It provides a gui that allows modifiying joint angles with sliders.
