Running ros_control demo

Steps overview

  • Create ros workspace/src
  • Git clone the control repo
  • Install necessary ros dependencies.
  • Source the ROS setup, build and source the workspace setup.
  • Run demo simulation

Steps


    cd ws1
    rosdep install -r --from-paths . --ignore-src -y    
    source /opt/ros/kilted/setup.bash
    colcon build --symlink-install
    source install/setup.bash
    ros2 launch ros2_control_demo_example_1 rrbot.launch.py
