Hardware resource

Function

The hardware resources realize communication to physical hardware and are exported as plugins. Each individual resource hardware is loaded at runtime and thus allows a flexible and dynamic composition of the setup to be controlled. The resource hardware is composed and configured solely through URDF.

Types

Actuator and System components have reading and writing capabilities (using the joint interface), whereas the Sensor component has only reading capabilities (using the sensor interface).
  • Actuator: Simple 1-DOF robotic hardware like motors or valves
  • System: Complex multi-DOF robotic hardware like industrial robots.
  • Sensor: Robotic hardware used for sensing the environment, either related to a joint (e.g., encoder) or a link (e.g., force-torque sensor).
