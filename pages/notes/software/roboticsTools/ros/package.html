Package creation

Use

[Automatic Addison]

Steps


    cd test/ros/
    mkdir src
    cd src/
    ros2 pkg create --build-type ament_cmake --license Apache-2.0 ros2_fundamentals_examples
    cd ..
    colcon build
    source install/setup.bash
    ros2 pkg list
To build a specific package: 

    colcon build --packages-select ros2_fundamentals_examples
Deleting a package: 

    cd src/
    rm -rf ros2_fundamentals_examples/
    cd ../
    rm -rf build/ros2_fundamentals_examples/ install/ros2_fundamentals_examples/
    colcon build
    source install/setup.bash
    ros2 pkg list | grep ros2_fundamentals*
