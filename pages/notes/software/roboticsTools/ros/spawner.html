Spawner node

Overview


    robot_controller_spawner = Node(
      package="controller_manager",
      executable="spawner",
      arguments=["Controller name defined in yaml", "--param-file", "path to file inc name"],
    )
Node(): This is a launch action that tells the system to run a specific process (a "node").

package="controller_manager": This specifies the package where the executable lives. The controller_manager is the "brain" that handles the lifecycle of controllers (starting, stopping, and switching them).
