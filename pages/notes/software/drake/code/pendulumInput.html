Namespace

The following defines a nested namespace structure. It indicates that the following code is part of the drake simulation framework and specifically an example for a pendulum system. 

namespace drake {
namespace examples {
namespace pendulum {

Uniform initialisation

Also called list or brace initialisation. 

PendulumInput{}.with_tau(0.0);
// is equivalent to

PendulumInput temp;
temp.with_tau(0.0);
Steps:
  • A temporary object of PendulumInput is created.
  • .with_tau(0.0) is immediately called on this temporary object.
  • The temporary object is discarded if the return value is not used.
