Cmake: Purpose

Overview


mkdir build; cd build

# Generate the build system files
cmake ..

# Compile the project
make
cmake generates the necessary build instructions from CMakeLists.txt that a specific build system like Make or Ninja understands to compile a project. The actual linking and compilation takes place in the make step.

Makefile

In the case of a Make build system, cmake generates a Makefile with instructions on how to:
  • Compile source files into object files.
  • Link those object files into an executable or library.
  • Handle dependencies between files, like ensuring that changes to header files trigger recompilation of dependent source files.
