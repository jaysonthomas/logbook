Git

Submodules

Ref. 

git submodule add https://github.com/RobotLocomotion/drake.git 'destinationFolder'
git submodule update --init --recursive

The 2nd command is used to initialise and update submodules present in git repositories that are cloned from elsewhere. Add the recursive option to do it recursively.

Unstage added files


git reset HEAD

Stash


# List all the stashes related to different branches with stash indices
git stash list

# Delete a stash
git stash drop index

# To show the files present in a stash. Add ' -p ' after show to see more details.
git stash show stash@{0}

# To create a stash entry with a description
git stash push -m "Description"

Git

  • Git branch
    • 
# List all branches
git branch -a

# Create branch Y from another, say X
# Changes from branch X will show up as changes from the master on the 
# new branch Y.
git checkout [X]
git checkout -b [Y]

# To merge Z into the newly created branch Y
git merge [Z]

# What branch am I on?
git branch --show
