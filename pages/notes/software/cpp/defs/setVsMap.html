Set vs Map

Set

It is a collection of unique keys with no associated values. Used when the need arises to check if something exists or to keep unique items.

Map

It is a collection of key-value pairs. It is similar to a dictionary or a lookup table. The key is normally looked up to find its corresponding value.

Unordered variants

The 2 are ordered by default. The unordered variants are hash-based and offer faster average-time lookups.
