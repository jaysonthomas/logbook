C++: Reference

Reference variables


int y = 10;
int &ref = y; // 'ref' is a reference to 'y'
The address of ref is made the same as that of y. Printing ref outputs 10. A reference is a way of creating an alias for an existing variable; the same data can be referred to using a different name.

Reference function parameters


# Following is the constructor for class Dog.
Dog(const std::string& name, const std::string& breed)

# Creating an instance of the class
Dog dog1("Rex", "German Shepherd");
Pass-by-value (where function parameters are of type std::string, no &) creates a copy of each string as it enters the function, before it even reaches the constructor body. That would 2 copies of the original string, if the string is also stored inside the class.

Passing a reference to an existing string avoids a copy entirely. const treats it as read-only. This is much faster, especially for long strings.
