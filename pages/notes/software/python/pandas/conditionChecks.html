Condition checks

Overview

Consider a dataframe with columns a,b,c,d. Based on some conditions (values of each row, of the columns), a new column status is populated. 

df['status'] = 'u'
df.loc[~df[['a', 'b', 'c', 'd']].any(axis=1), 'status'] = 'p'
df.loc[(df['b']) & ~df[['a', 'c', 'd']].any(axis=1), 'status'] = 'xd'
If a row has a b value that is True, but the remaining columns are False, then status = xd.

The assignment in the context of dataframes boils down to: 


df.loc[mask, 'status'] = 'p'
To mean: Select the status column, but assign p only for the rows where the mask is True.

.loc[] expects the first argument to be either:
  • a row label/index (e.g. 0 or 'row_id'), or
  • a boolean Series the same length as the DataFrame.
