Generating random gaussian noise

Overview

NumPy uses a pseudorandom number generator which is deterministic but appears random. It follows a specific algorithm that:
  • Starts with a seed — a starting value for reproducibility.
  • Generates uniform random numbers between 0 and 1.
  • Transforms those numbers into samples from a normal distribution (i.e. into normally-distributed values) using a mathematical method like Box-Muller or Ziggurat.
  • Scales and shifts the value using the mean and standard deviation provided.

x_t + np.random.normal(0.0, sigma)
