Mesh grid

Overview


x, y = np.mgrid[xAxisLimit[0]:xAxisLimit[1]:delta,    # Say, m elements
                yAxisLimit[0]:yAxisLimit[1]:delta]    # Say, n elements
  • x is an $m \times n$ grid where each column has $m$ values in an ascending order between the x limits with the specified step size. All columns are the same. x contains the x-coordinates, which change along the columns.

  • y is an $m \times n$ grid where each row has $n$ values in an ascending order between the y limits with the specified step size. All rows are the same. y contains the y-coordinates, which change along the rows.

Picking a corresponding element from each matrix, say x[1, 2] and y[1, 2], yields the coordinate pair (x[1, 2],y[1, 2]), which would be a point to do something with on the resultant grid.

Usage

It is used to calculate the pdf for each coordinate pair taken from the grid, the result of which can then be placed in another $m \times n$ matrix. 

stateSpace = multivariate_normal(mu, sigma)
stateSpace.pdf(configurationSpace)
