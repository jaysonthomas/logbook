Scipy: Solve IVP

Overview

[Open source github]

Solve initial value problem is a function in Python's SciPy library that helps solve differential equations.

RK45

The RK45 method uses an adaptive version of RK4 but with step-size control. 

def pendulum_ODE(t, y): 
  return (y[1], -g*np.sin(y[0])/l)

sol = solve_ivp(pendulum_ODE, [0, t_final], (theta0, theta_dot0), t_eval=np.linspace(0,t_final,t_final*fps+1))
Required arguments:
  • fun: Function that defines the system of ODEs: $\frac{dy}{dt} = f(t, y)$
  • t_span: Tuple specifying the time interval (t_start, t_end)
  • y0 Initial values of y (can be a list for multiple equations)
Optional arguments:
  • method
  • t_eval Specific time points where results are computed
  • args Extra arguments passed to fun
  • rtol Relative tolerance (error control)
  • atol Absolute tolerance
  • dense_output If True, returns a function to evaluate solution at any t
  • events Functions that stop integration when conditions are met
  • max_step Maximum step size
Output of the solver:
  • sol.t: Time points
  • sol.y: Solution values
