List: Remove entries

Overview

Given:
  • List fieldList of string entries
  • toRemoveFieldList containing partial string entries that if present in any of the fields, the particular fields are to be removed from the original list
Example: "timeInSeconds" should be removed from the original list if "time" is on the removal list.

Long code version


def removeFields(fieldList, toRemoveFieldList):
  output = []
  for field in fieldList:
    isBadFieldPresent = False
    for badField in toRemoveFieldList:
      if (badField in field):
        isBadFieldPresent = True
        break

    if (not isBadFieldPresent):
      output.append(field)

  return output

Condensed version


def removeFields(fieldList, toRemoveFieldList):
  return [
    field for field in fieldList
    if not any(badField in field for badField in toRemoveFieldList)
  ]
