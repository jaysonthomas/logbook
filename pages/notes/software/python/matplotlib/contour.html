Contour

Overview


plt.contour(gridX, gridY, pdf, contourLevels, colors=colour)
Given the multivariate probability density function values pdf for every point on an x,y grid, contour plots only those contour lines for specific pdf values specified by contourLevels.

The points of the X,Y grid is formed by combining the values corresponding to specific rows and columns from both the 2D matrices gridX, gridY.

Contour is a 2D representation of a 3D surface. It is like a topographic map:
  • The X and Y axes are the grid coordinates (like longitude and latitude).
  • The contour lines represent points of equal "height" or Z value (like altitude).
    • Here, z is the probability density calculated at each (x, y) point.
