Matplotlib in python: Animation

Overview


ani = animation.FuncAnimation(fig, animate, frames=10)

Arguments

  • fig - This is the figure where the animation will be drawn.
  • func - This is the function that updates the animation frame by frame.
  • frames - This decides how many frames (or steps) the animation will have. It can be a number or an iterable.
  • init_func - This is an optional function that sets up the initial state of the animation.
  • fargs - A tuple of extra arguments that you can pass to your update function.
  • interval - The time delay (in milliseconds) between frames.
  • repeat - A boolean that decides if the animation should loop.
  • repeat_delay - Time (in milliseconds) before the animation restarts when looping.
  • blit - A boolean that speeds up rendering (only works with some backends).
