Uninstall

Commands

Remove the actual software packages. The --purge flag ensures that configuration files are removed along with the binaries. 

sudo apt-get remove --purge drake-dev
Remove the Repository Source after checking for the file in the sources directory. 

ls /etc/apt/sources.list.d/
sudo rm /etc/apt/sources.list.d/drake.list
Remove any expired GPG key if apt complains about it. 

sudo apt-key del 82FAAFE5AFEF06D7
Tell Ubuntu to forget the old package data and update its package index/lists. 

sudo apt-get autoremove
sudo apt-get update

Extras

Bazel left behind large output files and caches in the home directory. 

rm -rf ~/.cache/bazel
If typing bazel still runs, it might have been installed via Bazelisk or by downloading a binary manually. Delete /usr/local/bin/bazel and ~/bin/bazel if they exist.
