Clion: Installation

Steps

Follow steps in CLion-2024.3.5/clion-2024.3.5/Install-Linux-tar.txt after the download.
  • Download
  • Move downloaded folder
  • ./clion.sh from bin/
  • Add to system path
  • Change heap size

Addition to system path

Add the following to line to the end of ~/.bashrc. sudo is not required. 

export PATH="/home/jayson/software/clion-2024.3.5/bin:$PATH"
Apply the changes 

source ~/.bashrc
Type clion in any directory.

Heap size change

Create clion64.vmoptions in the configuration directory ~/.config/JetBrains/CLion2024.3. Set the -Xms and -Xmx parameters.
  • -Xms: Sets the initial heap size (the minimum amount of memory allocated to the JVM at startup).
  • -Xmx: Sets the maximum heap size (the maximum amount of memory the JVM can use).

Check max heap size

It cannot exceed the system's total available free RAM.

Check Available System RAM


free -h
The "Total" value under the "Mem" row gives system's total RAM.

Check Current JVM Limits


java -XX:+PrintFlagsFinal | grep -i HeapSize
This will show various heap size limits set by default. For example, if MaxHeapSize is 8589934592, it means the JVM can use up to $\frac{8589934592}{1024^3} = 8$GB. By default, the JVM usually sets the maximum heap size to Â¼ of the total system memory.
