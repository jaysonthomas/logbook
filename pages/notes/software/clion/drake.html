Steps

[Ref]

  • Install the Bazel plugin.
  • Setup the Bazel wrapper.
  • Import Bazel project.
  • Configure a run or debug.

Bazel wrapper setup

[Ref]

This is a custom Bazel wrapper script written in python to setup any project specific requirements for the interaction of clion and bazel like configuring bazel with extra flags etc. It ensures the environment is set correctly in clion.

Configuration of a run or debug

  • Run -> Edit configurations
  • Target expression: //examples/pendulum:passive_simulation
  • Bazel command: run
  • Port: 7050
