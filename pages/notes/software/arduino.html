Installation

Creating a shortcut

Save the following in a file /home/jthomas/.local/share/applications/arduino.desktop. 

[Desktop Entry]
Type=Application
Name=Arduino IDE 2.3.2
Exec=/home/jthomas/Arduino/arduino-ide
In File-Preferences-Settings, set Sketchbook location to /home/jthomas/arduinoSketches. From Sketch-Include Library, download Rosserial arduino library. It should get downloaded to the selected sketches folder.
