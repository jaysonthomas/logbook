ROW_NUMBER

Overview


SELECT
  t2.id, t2.time, t2.error,
  ROW_NUMBER() OVER (
    PARTITION BY t1.xyz
    ORDER BY t1.startTime ASC
  ) AS rowIndex
FROM
  table1 AS t1
JOIN
  table2 AS t2
ON
  t1.id = t2.id
WHERE
  t2.time BETWEEN t1.startTime AND t1.stopTime
Since the join and filter operations happen first, every row of table 2 that meets the join + filter conditions have attached to them the corresponding row of table 1.

The PARTITION BY clause can then group rows together based on the duplicated fields and assign row numbers. Different groups can thus be created with their rows indexed.
