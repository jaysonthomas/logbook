Quantiles

Overview


SELECT
  APPROX_QUANTILES(error, 3) AS errorQuantile,
  rowIndex
FROM
  table1
GROUP BY
  rowIndex
Assume rowIndex is obtained by using ROW_NUMBER().

Building up quantiles is like building up a fence - to create 3 sections of fence, 4 fence posts are required. In data, these fence posts are the quantile values.

If there are 6 values say in a table say with rowIndex = 1, with a request for 3 quantiles, the result would actually be 4 quantiles:
  • 0%: is just the minimum value.

  • First quantile (~33%): With 6 data points, the value at the 33rd percentile is found at position 6 * 0.333 = 2

  • Second quantile (~67%): With 6 data points, the value at the 67th percentile is found at position 6 * 0.667 = 4

  • 100%: is just the maximum value.
