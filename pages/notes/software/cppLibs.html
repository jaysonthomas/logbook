C++ Libraries

Eigen

  • Download zip file from here. Downloaded eigen-3.4.0.

  • INSTALL has instructions to: 
    
mkdir build; cd build
cmake ../     

# The above command will print options to the terminal that 
# can be given to cmake.
sudo make install

    The cmake step will set the install location of the Eigen headers as
    CMAKE_INSTALL_PREFIX/INCLUDE_INSTALL_DIR where
    • CMAKE_INSTALL_PREFIX is /usr/local
    • INCLUDE_INSTALL_DIR is include/eigen3

    Change the install location using: 
    
cmake . -DCMAKE_INSTALL_PREFIX=yourprefix
# Or
cmake . -DINCLUDE_INSTALL_DIR=yourdir

  • Including an Eigen header in code
    • 
#include <eigen3/Eigen/Dense>

// If compiling using g++ test.cpp -I/usr/include/eigen3/ -o output
#include <Eigen/Dense>

Using python headers in code

Usage: 

#include <Python.h>
#include <numpy/arrayobject.h>
These libraries are found in /usr/include/python3.10/. To be able to link them: 

sudo apt-get install python3.10-dev
g++ test.cpp -I/usr/include/python3.10/ -lpython3.10 -o output
