Control part

The C++ simulator is more bare-bones than the Python/Unity simulator, but it's more realistic in the physics that it models. We'll be porting code written in python. The aim of the project is to:
  • Setup the C++ simulator and the development environment
  • Familiarisation with tools to evaluate the controller's performance
