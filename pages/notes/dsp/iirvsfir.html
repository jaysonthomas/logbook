IIR vs FIR

IIR is called infinite because the current filter's output depends on both the current input value and the previous output value(s) (just 1 or a group). The previous output value(s) depends on past inputs and outputs which can be traced back to the 1st input value.

FIR is finite because it only depends on the weighted value of the current input. Good ref.
