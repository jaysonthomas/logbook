Background knowledge

This seems like a good starting point using matplotlib and pytransform3d. Had to install the latter library using pip. There are good resources in google on it.
  • Transformation matrices
    • Assume we know the three Euler angle values of frame B w.r.t. frame A and the translation $(tx, ty, tz)$ of the origin of frame B from frame A in the frame A coordinates. The transformation matrix for converting from the frame of reference B to A is given as $T_{AB}$.

  • Multiplication order
    • It's from right to left. The following equation takes a point $p_0$ from frame of reference D to frame of reference A. Note that a point in homogenous coordinates is represented as $[x, y, z, 1]$ $$T_{AB} = \begin{bmatrix} R & t\\0 & 1 \end{bmatrix}$$ $$T_{AD} = T_{AB}T_{BC}T_{CD}$$ $$^AP_0 = T_{AD}^DP_0$$
