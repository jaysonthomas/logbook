Measurement model

Overview

The effect of a coriolis force can be measured. It can then be used to infer the rotational speed of a rotating reference frame since $F_c$ is a function of $\omega$: $$ |F_c| = f(\omega) $$ The value (e.g., rotation rate) that a sensor (e.g., gyroscope) reports is unlikely to be the correct one. The role of a measurement model of a sensor is to help make sense of the imperfect measurements. It includes a measure of how accurate a sensor's measurements are; in this case, that of the inferred quantity $\omega$.

Details

A simple measurement model for a gyroscope: $$\begin{align*} \tilde{\omega} &= \underbrace{\omega}_{\text{True angular} \atop \text{velocity}} + \underbrace{b}_{\text{Bias}} + \underbrace{\eta}_{\text{Additive noise}} \\\\ \eta &\sim N(0, \sigma^2_{gyro}) \end{align*}$$ The measured rotation rate $\tilde{\omega}$ is equal to the true angular velocity, corrupted by some:
  • Bias
    • It needs to be identified ahead of time. It might be temperature dependent and might change slowly, but it's something that stays constant at least for some amount of time.

  • Zero mean gaussian noise
    • It changes for every measurement.

Attitude estimation

Gyro measurements can be used to estimate the body rate of a vehicle, which can then be integrated to get the actual attitude. $$ \theta_t = \theta_{t-1} + \tilde{\omega}\Delta{t} + \underbrace{\epsilon}_{\text{Approximation} \atop \text{error}} $$ Since, computing an integral is a discrete approximation, in reality, there would also be an approximation error which can be ignored for now. It can also be assumed that any potential bias is calibrated and removed from the Gyro measurements. The equations become: $$\begin{align*} \tilde{\omega} &= \omega + \eta \\ \theta_t &= \theta_{t-1} + \tilde{\omega}\Delta{t} \end{align*}$$ The initial angle $\theta_0$ would need to be known. This type of navigation-by-integration is called dead reckoning. Inertial navigation, that is navigation based on inertial sensors only, is purely based on dead reckoning. They are highly susceptible to drift.

Drift

A gyroscope measures angular velocity. The orientation is computed by integrating tiny slices of velocity over time. Because of the noise and bias in each measurement, the tiny error gets added over and over again leading to drift.

Gyroscopes, however, don't suffer from jitter and can provide immediate, smoother angle measurements. An accelerometer is used to slowly nudge the gyro back to reality whenever it starts to drift away from the gravity vector.
