Point cloud

Point cloud data

Lidar data is stored in a format called PCD. A .pcd file is a list of (x,y,z) cartesian coordinates along with intensity values. Each point is of the format: x coord, y coord, z coord, i.

Tools

Tools are needed to:
  • filter, cluster and segment objects
  • separation of the ground plane
The point cloud library and tools within opencv can be used.
