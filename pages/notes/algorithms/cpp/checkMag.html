using namespace std;

void checkMagazine(vector magazine, vector note) {
    unordered_map wordCount;

    // Count each word in the magazine
    for (const string& word : magazine) {
        ++wordCount[word];
    }

    // Try to build the note
    for (const string& word : note) {
        if (wordCount[word] > 0) {
            --wordCount[word];
        } else {
            cout << "No" << endl;
            return;
        }
    }

    cout << "Yes" << endl;
}

Time complexity analysis

O(n + m).

In the first for loop, for a word length of m (magazine), each map operation is O(1) on average using unordered_map. Since the loop runs m times, it is O(m). For the 2nd for loop, for a word length of n (note), each lookup and decrement in the hash map is O(1) on average. Since the loop runs n times, it is O(n).

For a nested loop, it is O(n * m).

Space complexity analysis

In the worst case, if all words in magazine are unique, the complexity is O(m).
