Differential equations

Definition

A differential equation is a general term that refers to any equation involving derivatives of a function. It describes how a particular function (continuously) changes based on its derivatives (i.e. successive values).

In the real world, many physical parameters are related to each other through differential equations.

Types

Ordinary DE

It is a differential equation that involves derivatives with respect to only one independent variable

For LTI systems, physical parameters are related to each other strictly through ODEs.

Partial DE

It involves partial derivatives with respect to multiple independent variables (e.g., both time and position).

Discrete time equivalent

In discrete time, a mathematical expression that relates the difference between successive terms in a sequence is called a difference equation. $$ y(n) = y(n-1) + 3 $$ This is a first-order linear difference equation where each term is 3 more than the one before it.
