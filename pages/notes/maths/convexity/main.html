Convexity

Convex set

A convex set is one in which a line connecting any 2 points in the set is fully contained in the set.

Examples

  • Linear subspaces ($Ax= b$, hyperplanes)
  • Half space/box/polytope ($Ax \leq b$)
  • Ellipsoids

  • Cones
    • This is a 2nd order cone. It shows up as friction cones in contact scenarios. Some more detail in lec 10.

Convex function

A function whose epigraph is a convex set. Epigraph is the portion above a curve. This function takes in a vector and outputs a scalar similar to an objective function. $$ f(x) \; : \; \mathbf{R}^n \rightarrow \mathbf{R} $$
