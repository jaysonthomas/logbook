Overview

A Jacobian matrix represents the first-order partial derivatives of a system of differential equations.

Linear systems

For a linear system such as $\dot{x} = Ax$, the Jacobian is simply the matrix $A$. In this specific case, it is constant for all values of $x$.

Nonlinear systems

For a nonlinear system $\dot{x} = f(x)$, the Jacobian is a function of $x$.

For example, if: $$\begin{align*} \dot{x}_1 &= x_1 x_2 \\ \dot{x}_2 &= x_1^2 - x_2 \end{align*}$$ The Jacobian will contain variables (like $x_1$ and $x_2$), meaning its values change as the system moves.
