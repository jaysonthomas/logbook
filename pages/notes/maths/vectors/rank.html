Matrix rank

Full row rank

A matrix is full row rank if its rows are linearly independent i.e. the matrix cannot be expressed as a linear combination of other rows. If a matrix is not of full row rank, it means that there exists a linear dependence among its rows.

The rank of a matrix is the maximum number of linearly independent rows or columns in it. If the rank is less than the number of rows ($m$), then the matrix is not of full row rank - there exists at least one row that can be expressed as a linear combination of the other rows.

For an inverse dynamics problem: $$ \dot{q} = J^{-1} \dot{x} $$ $J \in \mathbb{R}^{n \times m}$ where the end-effector has $n$ DOF and there are $m$ joints. If $m < n$, then the rank of $J$ can only be $m$ since few ($n-m$) of the $n$ rows will be dependent on the others. In this case, since $J$ is not full row rank, there can be configurations in the operational space for which there is no solution in the joint space, i.e. $J$ becomes non-invertible.
