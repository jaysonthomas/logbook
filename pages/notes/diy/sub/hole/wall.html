Hole in a wall

Plasterboard

  • If it's a plasterboard, how thick is it?
  • If drilled through the plasterboard, is there a hard wall behind? If yes, at what distance from the plasterboard?
  • What is the hard wall behind made of - steel, concrete or brick?

Brick

[Charlie DIY]

A normal drill can be used if the bricks are not too hard. Use a masonry drill bit (carbide tip).
  • Create a pilot hole with a hammer and screw.
  • Create an initial hole using the screw action.
  • Using the hammer function, continue drilling.
  • Use dust pan to collect dust. Note what material it is.
